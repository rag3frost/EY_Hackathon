# AutoAide AI

AutoAide AI is a modern, futuristic vehicle diagnostics dashboard built with React, Tailwind CSS, and Three.js. It features a realistic 3D car visualization, an AI-powered repair assistant, and a comprehensive suite of diagnostic tools.

## Features

- **Interactive 3D Car View**: Visualize your vehicle in 3D with realistic lighting and controls (Rotate, Zoom).
- **AI Repair Assistant**: Chat with an intelligent AI to diagnose issues and get repair advice.
- **Dynamic Car Selection**: Choose your vehicle model (Sedan, Sports, SUV) and see the 3D model update in real-time.
- **Repair History**: Track past services and maintenance records.
- **Live Assistance**: Connect with certified mechanics via voice or video.
- **Secure Authentication**: User registration and login flow with data persistence.
- **Responsive Design**: Fully responsive UI with a premium glassmorphism aesthetic.

## Tech Stack

- **Frontend**: React, Vite
- **Styling**: Tailwind CSS, Lucide React (Icons)
- **3D Graphics**: Three.js, React Three Fiber, Drei
- **State Management**: React Context API

## Getting Started

1.  **Clone the repository**
    ```bash
    git clone <repository-url>
    cd auto-aide-ui
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```

## Usage

1.  **Register/Login**: Create an account and select your vehicle model.
2.  **Dashboard**: Explore the 3D view and use the sidebar to navigate.
3.  **AI Chat**: Type your car troubles in the chat box to get instant feedback.
4.  **Settings**: Configure your preferences and API keys.
